### Como usar:




Caso queira rodar localmente e testar/adicionar novas funcionalidades basta:

```git
$ git clone <LinkDesteRepositorio/>
```
e então, para npm:
```js 
$ npm start  
```
e para yarn:
```js
$ yarn Start
```
